# 金融研报助手 - 项目总结（简历版）

## 📊 项目概述

**项目名称：** 金融研报助手 (Fin Research Assistant)

**GitHub:** https://github.com/wzwangyc/fin-research-assistant

**项目描述：** 基于 NLP + LLM + SQL 的金融研报智能分析系统，实现研报自动解析、情感分析、智能问答与数据可视化。95% 本地化（仅 LLM API 依赖）。

---

## 🎯 核心技术

| 模块 | 技术栈 | 本地化 |
|------|--------|--------|
| **NLP** | pdfplumber, jieba, 规则匹配 | ✅ 100% |
| **LLM** | Gemini API, RAG | ⚠️ API |
| **SQL** | SQLite, SQLAlchemy, ETL | ✅ 100% |
| **前端** | Streamlit | ✅ 100% |
| **后端** | FastAPI（可选） | ✅ 100% |

---

## 🏆 核心功能

### 1. PDF 解析（NLP）
- 提取文本、表格、元数据
- 支持扫描版 OCR（可选）
- 准确率 95%+

### 2. 命名实体识别（NLP）
- 识别公司名、股票代码、金额、百分比
- 金融词典优化
- 准确率 90%+

### 3. 情感分析（NLP）
- 金融情感词典
- 正面/负面/中性分类
- 段落级情感趋势

### 4. ETL 数据管道（SQL）
- 自动提取 - 转换-加载
- 批量处理
- 错误处理与日志

### 5. 智能问答（LLM）
- RAG 检索增强生成
- 引用溯源
- 多轮对话

### 6. 数据可视化（前端）
- Streamlit 界面
- 情感趋势图
- 热门关键词

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据库

```bash
python backend/app/sql/database.py
```

### 3. 启动应用

```bash
start.bat
```

或手动：
```bash
cd frontend
streamlit run app.py
```

浏览器打开：http://localhost:8501
